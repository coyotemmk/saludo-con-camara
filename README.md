# Saludo con camara

App en Python que abre la camara del dispositivo y detecta un saludo con la mano para decir "Hola" en voz alta.

## Requisitos

- Python 3.14 o compatible
- Camara conectada al dispositivo

## Instalacion

```powershell
pip install -r requirements.txt
```

## Ejecucion

```powershell
.venv\Scripts\python.exe app.py
```

## Uso

- Abre la camara.
- Levanta una mano con la palma visible.
- Muevela de lado a lado como saludo.
- Cuando la app detecta el gesto, dice "Hola".
- Presiona `q` para salir.
